# Sample Input Video

This folder provides a short representative sample input video clip for demonstrating the runtime input format of the UAV surveillance pipeline.

## File

```text
sample_input_video.mp4
```

## Purpose

The sample video is provided only for code-execution demonstration and pipeline-format verification.

It does **not** correspond to the full evaluation videos used for the quantitative runtime results reported in the manuscript.

## Availability of Full Experimental Materials

The full author-constructed UAV image dataset, trained model weights, TensorRT engine files, and full evaluation videos are not publicly distributed due to institutional and data-use restrictions.

These materials are available from the corresponding authors upon reasonable request, subject to institutional and data-use restrictions.
